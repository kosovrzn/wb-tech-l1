# L1.11 - Пересечение множеств

Пересечение двух неупорядоченных множеств (слайсов) целых чисел. В результате остаются только элементы, присутствующие одновременно в обоих исходных слайсах, без повторов.

## Запуск

```bash
cd l1.11
go run .
```

Пример вывода:

```
A = [1 2 3 5 5 10 15]
B = [2 3 4 5 5 8 10]
Intersection = [2 3 5 10]
```
