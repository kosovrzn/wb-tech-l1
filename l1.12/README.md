# L1.12 - Собственное множество строк

Создать множество уникальных слов для последовательности `("cat", "cat", "dog", "cat", "tree")`.

Программа выводит исходный список и множество уникальных слов (отсортировано для удобства чтения), например: `Unique set: [cat dog tree]`.

## Запуск

```bash
cd l1.12
go run .
```
