# L1.8 - Манипуляции битами

В этом задании нужно изменить конкретный бит числа типа `int64`, используя побитовые операции `|` и `&^`.

## Как запустить

```bash
cd l1.8
go run . -n 5 -bit 0 -value 0
```

Параметры командной строки:

- `-n` — исходное число (`int64`).
- `-bit` — позиция бита, который нужно изменить (0 — младший бит, максимум 63).
- `-value` — значение битa (`0` очищает бит, `1` устанавливает).

Пример выше повторяет условие: начиная с `5 (0101b)` и обнуляя бит `0`, получаем `4 (0100b)`.
