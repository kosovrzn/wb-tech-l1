# L1.10 - Группировка температур

Группируем значения `-25.4, -27.0, 13.0, 19.0, 15.5, 24.5, -21.0, 32.5` по диапазонам с шагом 10 градусов: -20 включает числа от -20 до -29.9, 10 — от 10 до 19.9 и т.д.

## Запуск

```bash
cd l1.10
go run .
```

## Пример вывода

```
-20: [-25.4 -27 -21]
10: [13 19 15.5]
20: [24.5]
30: [32.5]
```
