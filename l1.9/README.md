# L1.9 - Конвейер чисел

Программа строит двухступенчатый конвейер с горутинами:

1. Первая горутина читает числа из массива и отправляет их в канал `source`.
2. Вторая горутина читает из `source`, умножает числа на 2 и пишет результат в канал `doubled`.
3. Главная горутина выводит содержимое `doubled`, корректно завершая чтение после закрытия канала.

## Как запустить

```bash
cd l1.9
go run .
```

На экране появятся все числа исходного массива, умноженные на два.
